- **ICC cybersecurity** — security must be part of shared backbone, not an afterthought

### Analysis

The baseline quantifies the cost of fragmentation; ICC demonstrates existing shared-service maturity. The proposal phase should map **category-by-category** (cloud, network, office, systems) which functions migrate to common providers vs. remain entity-specific for good reason.
